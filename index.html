# Fluxo - Controle de Combustível Aplicativo simples e eficiente para controlar gastos com combustível. ### ✨ Funcionalidades - Registro rápido de abastecimentos - Histórico completo com filtros - Gráficos de consumo e custo - Editar e excluir lançamentos - Suporte a Etanol e Gasolina - Funciona offline (PWA) ### 🚀 Como usar 1. Acesse: [https://danilolima-cyber.github.io/Fluxo-app/](https://danilolima-cyber.github.io/Fluxo-app/) 2. Clique em "Adicionar à tela inicial" no navegador 3. Pronto! Use como um app normal ### Tecnologias - HTML, CSS e JavaScript puro - Firebase Realtime Database - Chart.js --- **Desenvolvido com ❤️**
